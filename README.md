## 📖 Overview

The user requested to **change the entire frontend** to exactly match the Google Chrome Dinosaur offline game. The speed, elements, and exact look have been perfectly replicated by utilizing the original assets and offline runner source code.

**No frameworks. Pure HTML5 + CSS3 + Vanilla JavaScript.**

---

## 🎮 How to Play

| Action | Keyboard |
|--------|----------|
| Jump | `Space` or `↑ Arrow` |
| Duck | `↓ Arrow` |
| Start Game | `Space` |
| Restart | `Space` |

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/algorithnicmind/web-space-dragon.git
   ```
2. **Install and Run:**
   ```bash
   npm install
   npm run dev
   ```
3. Open `http://localhost:5173/` in your browser.

4. **Play!** Press `Space` to start.
